# Creative Studio | Design & Development Agency

A professional portfolio website for a creative studio, featuring brand identity, web development, and digital marketing services.

## Technologies Used

- **Vite** - Build tool
- **React** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **framer-motion** - Animations
- **shadcn/ui** - Component library

## Getting Started

1.  **Install dependencies**:
    ```sh
    npm install
    ```

2.  **Start development server**:
    ```sh
    npm run dev
    ```

3.  **Build for production**:
    ```sh
    npm run build
    ```

## Project Structure

- `src/components` - Reusable UI components
- `src/pages` - Application pages
- `src/assets` - Static assets like images and logos
- `public` - Public assets (favicon, robots.txt)
